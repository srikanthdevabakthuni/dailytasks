# TASK 2

 ## Create / Configure Features

### Create Posts
1. Go to Dashboard → Posts → Add New  
2. Enter title and content  
3. Add category and tags  
4. Click Publish  


### Create Pages
1. Go to Dashboard → Pages → Add New  
2. Add title and content  
3. Click Publish

### Upload Media
1. Go to Dashboard → Media → Add New  
2. Upload images/files  
3. Insert media into posts/pages  


## Task 3: Customize UI / Settings

### Customize Posts
- Add featured image  
- Use categories and tags  


### Customize Pages
- Use page templates  
- Customize layout   



### Customize Media
- Add alt text (SEO)  
- Edit image (crop, resize)  
- Organize media files  



## Task 4: Debug / Optimize

###  Common Issues

### Media Not Uploading
- Check file size limit  
- Verify file type

### Posts Not Displaying
- Check publish status  
- Verify categories  


### Pages Not Opening
- Check permalink settings  
- Re-save permalinks  

###  Optimization Tips  

- Compress images before upload  
- Use proper categories and tags  
- Avoid duplicate content
