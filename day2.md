#  Hosting, XAMPP, WordPress Installation

##  Task 1:  Concepts

###  Hosting
Hosting is a service that stores website files and makes them accessible online.

###  XAMPP
XAMPP is a local server that includes:
- Apache
- MySQL
- PHP

###  WordPress
WordPress is a CMS used to build websites easily.
