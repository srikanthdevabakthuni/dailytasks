Plugins add extra functionality to a WordPress website.

# Task 2: Configure / Configure Features
## Install Theme
1. Go to Dashboard → Appearance → Themes  
2. Click Add New  
3. Search for a theme 
4. Click Install → Activate  


##  Install Plugins
1. Go to Dashboard → Plugins → Add New  
2. Search for plugins  
3. Click Install → Activate

 # Task 3: Customize UI / Settings

##  Customize Theme
- Go to Appearance → Customize  
- Change:
  - Logo  
  - Colors  
  - Fonts  
  - Header & Footer  
## Plugin Customization
- Configure plugin settings  
- Enable/disable features  
- Customize forms/layout  


#  Task 4: Debug / Optimize

##  Common Issues

### Theme Not Working
- Check compatibility with WordPress version  
- Clear cache  


### Plugin Conflict
- Deactivate plugins one by one  
- Identify conflicting plugin  



##  Optimization Tips

- Use trusted themes/plugins  
- Update regularly  
- Remove unused plugins  
- Optimize performance
