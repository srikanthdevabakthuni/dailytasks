#  Task 2: Create / Configure Features

##  Dashboard Setup
- Login: `/wp-admin`  
- Explore menu options  
- Update profile settings  


##  Configure Permalinks
1. Go to Settings → Permalinks  
2. Select:
   Post Name  
3. Click Save Changes  


## Create Categories
1. Go to Posts → Categories  
2. Add:
   - Name  
   - Slug  
3. Click Add Category  


##  Create Tags
1. Go to Posts → Tags  
2. Add:
   - Name  
   - Slug  
3. Click Add Tag
