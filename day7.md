# Task 2: Create / Configure Features

## Install & Activate Theme
1. Go to Dashboard → Appearance → Themes  
2. Click Add New  
3. Search for a theme (e.g., Astra)  
4. Click Install → Activate  


## 🔧 Configure Theme Settings
- Go to Appearance → Customize  
- Modify:
  - Site title & tagline  
  - Logo
  - Colors & typography
