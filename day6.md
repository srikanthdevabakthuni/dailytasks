

#  Task 1: Implement Core Concept – Build Blog Website

A blog website is a platform where users can publish articles (posts) and organize them using categories and tags.

## Features of Blog Website:
Create and publish posts  
Categorize content  
 Add tags for better search   

## Implementation:
Installed WordPress using XAMPP  
Created a blog site using a theme  
 Used posts for blog content  


# Task 2: Create / Configure Features

## Create Blog Posts
Go to Dashboard → Posts → Add New  
  Add:
   Title  
   Content  
   Category  
   Tags  
 Click Publish  


##  Categories Setup
Go to Posts → Categories  
 Create categories:
   Technology    


##  Tags Setup
Add relevant tags to each post  
  Example:
   WordPress  


## 🔗 Permalink Configuration
- Go to Settings → Permalinks  
- Select **Post Name**  


# Task 3: Customize UI / Settings
